Codigo Nome Expressao Descricao
ULA e a unidade logica e aritmetica: recebe A, B e um codigo de operacao. Interface envia sinais pela API serial e tambem le botoes/LEDs fisicos nos snapshots. EEPROM recebe um log quando uma operacao e executada pelo OK ou pela UI.

Registradores

Portas B / C / D

DDR define se cada pino trabalha como entrada ou saida. PORT mostra o nivel escrito nos pinos de saida ou pull-up nas entradas. PIN mostra o nivel eletrico lido no momento, incluindo botoes fisicos.

Timer0 / Timer1 / Timer2

Temporizadores

TCNT sao contadores internos dos timers do ATmega328P. TCCR sao registradores de controle dos timers. Leitura e apenas observacao: a interface nao reconfigura os timers.

Dump ao vivo

SRAM ula_probe[128]

0x00
SRAM e a memoria viva do programa. A janela `ula_probe` expõe 128 bytes instrumentados. EEPROM guarda o historico circular das operacoes mesmo apos desligar. FLASH mostra uma janela diagnostica da memoria de programa.
